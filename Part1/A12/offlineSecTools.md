## Discover 5 unique offline security tools

##### Safe
- Safes are used to provide physical access control to items or physical information.
- With a code is ensures only authorised individuals have access to what is inside, preventing theft and ensuring the security of its contents.
- I discovered this safe at my job, where only the site manager knows the code.

##### Security Camera
- Security cameras are a very commonplace security measure to ensure that sensitive locations are watched at all times, even if not actively.

##### Physical Locks
- Physical locks are an extremely common offline security tool used to restrict access to certain areas. There is another portfolio point about locks, so this one is unique to demonstrate their role as security tools.
- Depending on the lock, one might need a key, code or biometric to be authorised and gain access to an area behind a lock.
- This one is also on the 3rd floor of Reid Library, and I have absolutely no idea what is behind it, so it is doing it's job well.

##### Smoke Detector
- Smoke detectors are an offline security measure that detect smoke and other signs of fire in an area. They may be connected to further systems by a network, especially in a university, but the detector itself has an alarm that will sound to alert inhabitants of danger.
- For this reason they are often battery powered, so that if there are network or power issues they are not affected.

##### Signage
- Signs or warnings with information regarding the item, area or object they concern are also common security measures.
- While simple and not always effective, they indicate that only authorised individuals may access an area or that there is danger ahead.
- This information supports security policies in place and informs users of them.
