## Discover 5 AI-enabled security solutions

##### AI Fraud Detection
- AI based fraud detection has been implemented by many businesses like Paypal to analyse transaction data.
- Location, patterns of spending and behaviour of each user is tracked to identify if suspicious activity is detected.
- AI is used to learn what a normal purchase or transaction looks like and then flag and prevent unusual ones.
- I discovered this by researching the uses of AI in defensive cybersecurity
- Link: IBM (https://www.ibm.com/topics/fraud-detection)

##### Neural Networks with Facial Recognition
- Face ID and other facial recognition programs are commonly used as a security measure now, like using a facial scan to unlock a device.
- Many of these programs, such as Face ID, are powered by neural networks, an AI solution to security challenges like passwords.
- I discovered the use of neural networks through my knowledge of facial recognition and some confirming research.
- Link: Microsoft Azure (https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-face-recognition#:~:text=Face%20recognition%20is%20a%20form,identify%20or%20group%20a%20face.)
