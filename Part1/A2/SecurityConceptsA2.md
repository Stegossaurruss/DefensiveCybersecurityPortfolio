## Discover security concepts used in public space

##### Motion-based home security cameras
- My house has a few of these set up around it, especially covering all entrances.
- They automatically upload footage to a cloud server when motion is detected.
- I discovered this while checking out the security features of my house.

##### Ring Home Security Doorbell
- This doorbell has a camera, also designed to sense motion and will also record upon the doorbell being rung.
- It can send an alert to mobile devices, and be configured to alert a user if suspicious motion is detected (at strange times, etc.)
- I discovered this while checking out the security features of my house.

##### Pinpad and biometric door lock
- I left this lock out of my other lists for inclusion here, it has a pinpad based code unlocking system, as well as a fingerprint scanner.
- Each member of my family has a different code, so the user can see who has accessed the door, or whose code was used.
- It has the added bonus of not requiring a key, which means it cannot be misplaced into the hands of a potential threat actor.
- I also discovered this searching for security measures in my home.
